# Final Sweep & Consistency Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** As the FINAL plan after every screen has migrated to the new component library, delete all now-unreferenced legacy components and dead code (each deletion gated on a grep proving zero references), run an app-wide accessibility/consistency sweep (no `.uppercase()`, no developer-string content descriptions, ≥48dp interactive targets, no stray hardcoded `Color(0x…)`/off-grid `.dp` in `ui/`), and verify a full Android build + install plus an iOS source-set compile.

**Architecture:** This plan is deletion-and-verification only — it writes no new components and re-implements nothing. Every deletion is a two-step ritual: (1) run an exact grep that must return zero non-declaration references, (2) only then delete the now-orphan declaration. The consistency sweep is a set of exact grep "checklist" commands whose expected result is zero hits (or an explicitly enumerated allowlist), followed by a clean compile of both targets. It runs LAST because every gate assumes plans 04–10 and 12 have already migrated all call sites onto the §5 component APIs.

**Tech Stack:** Kotlin Multiplatform, Compose Multiplatform, Material3, Koin, Compose Resources. All shared UI in commonMain (KMP/iOS-safe).

**Sprint:** S4 · Account & Finish · **Plan 11 of 11** (authored last; runs after Plan 12 Map too — see Depends on).

**Depends on:** Plans 04–10 (all screen migrations) AND Plan 12 (Map screen). Also transitively Plan 01 (tokens), Plan 02/03 (components) since the new APIs replace the legacy ones. **This plan MUST run dead last.** If any earlier plan is incomplete, the deletion gates below will (correctly) find lingering references and the corresponding deletion task must be skipped — do not force a deletion that still has callers; instead flag the owning screen plan as incomplete.

- [ ] **Step 1: Create the test in `androidUnitTest`.** Write `shared/src/androidUnitTest/kotlin/com/mobileprism/fishing/ui/sweep/LegacyComponentRemovalTest.kt` with exactly:

```kotlin
package com.mobileprism.fishing.ui.sweep

import java.io.File
import kotlin.test.Test
import kotlin.test.assertTrue

class LegacyComponentRemovalTest {

    private val uiRoot = File("src/commonMain/kotlin/com/mobileprism/fishing/ui")

    private val bannedSymbols = listOf(
        "fun DefaultButton(",
        "fun DefaultButtonFilled(",
        "fun DefaultButtonOutlined(",
        "fun DefaultButtonSecondaryLight(",
        "fun LoadingIconButtonOutlined(",
        "fun BigText(",
        "fun HeaderTextSecondary(",
        "fun PrimaryTextBold(",
        "fun SecondaryTextLight(",
        "fun MyCardNoPadding(",
        "fun MyCard(",
        "fun MyClickableCard(",
        "fun DefaultCard(",
        "fun DefaultCardClickable(",
        "fun NoContentView(",
        "fun NoInternetView(",
        "fun ErrorView(",
        "fun FishAmountAndWeightView(",
        "fun LottieStars(",
        "fun MapLayersButton(",
        "fun MapSettingsButton(",
    )

    @Test
    fun legacyComposableDeclarationsAreRemoved() {
        if (!uiRoot.exists()) return
        val ktFiles = uiRoot.walkTopDown().filter { it.isFile && it.extension == "kt" }.toList()
        val offenders = mutableListOf<String>()
        for (file in ktFiles) {
            val text = file.readText()
            for (banned in bannedSymbols) {
                if (text.contains(banned)) {
                    offenders.add("${file.path} declares $banned")
                }
            }
        }
        assertTrue(
            offenders.isEmpty(),
            "Legacy composables must stay deleted, found:\n" + offenders.joinToString("\n")
        )
    }
}
```
