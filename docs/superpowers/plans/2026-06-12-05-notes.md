# Notes (Tabs, Catches, Places, Statistics) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor the Notes container and its three tabs (Catches, Places, Statistics) onto the shared design-system components — `AppTabRow`/`TabbedPager`, `PagedListScaffold`, `StatTile`, `ChartCard`, `SortOptionsSheet`, `EmptyState`/`ErrorState` — eliminating the near-verbatim Catches/Places duplication, replacing the raw primary `TabRow`, brand-tinting the charts, and fixing the entangled correctness/i18n bugs (uncontrolled `setSortOrder`, hardcoded English month labels, dead `Filter` branch, missing a11y descriptions).

**Architecture:** Notes screens are CONSUMERS of the shared component library built in Plans 01–03; this plan shows call sites and screen edits, never re-implements shared components. The Notes container swaps its raw `TabRow` for `AppTabRow`/`TabbedPager` (surface-colored, real selected/unselected states) and its duplicated sort sheets for one generic `SortOptionsSheet`. `UserCatchesScreen` and `UserPlacesScreen` both collapse onto a single shared `PagedListScaffold<T>` (append-error footer, end-of-list, FAB bottom inset via `Spacing.fabClearance`, `animateItem()`, skeleton→content crossfade, true-centered empty/error). `StatisticsScreen` swaps its private `StatCard`/`ChartSection`/`BarChartView` for shared `StatTile`/`ChartCard`, its month axis labels for a new KMP-safe localized month formatter, and its hand-rolled state ladder for `ScreenStateContent`.

**Tech Stack:** Kotlin Multiplatform, Compose Multiplatform, Material3, Koin, Compose Resources. All shared UI in commonMain (KMP/iOS-safe).

**Sprint:** S2 · Primary Screens (Phase C2 — Notes) · **Plan 05 of 11**

**Depends on:** Plan 01 (Foundation tokens: `Spacing`, `Elevation`, `Shape`, `Motion`, `Emphasis`, `Type`), Plan 02 (Core component library: `EmptyState` + presets, `ErrorState`, `LoadingState`, `ScreenStateContent`, `PagedListScaffold`, `ListAppendLoader`, `SkeletonBox`/`SkeletonLine`, `AppTabRow`/`TabbedPager`, `SortOptionsSheet`, `BrandFab`, `StatTile`, `ChartCard`, `IconStatChip`/`CountBadge`). If a referenced component is not yet present when a task runs, STOP and complete the owning plan first — do **not** re-implement it here.

  private val monthResources: List<StringResource> = listOf(
      Res.string.month_jan, Res.string.month_feb, Res.string.month_mar,
      Res.string.month_apr, Res.string.month_may, Res.string.month_jun,
      Res.string.month_jul, Res.string.month_aug, Res.string.month_sep,
      Res.string.month_oct, Res.string.month_nov, Res.string.month_dec,
  )

  fun monthShortLabelResource(yearMonthKey: String): StringResource? {
      val parts = yearMonthKey.split("-")
      if (parts.size != 2) return null
      val month = parts[1].toIntOrNull() ?: return null
      return monthResources.getOrNull(month - 1)
  }

  @Composable
  fun monthShortLabel(yearMonthKey: String): String {
      val resource = monthShortLabelResource(yearMonthKey) ?: return yearMonthKey
      return stringResource(resource)
  }
